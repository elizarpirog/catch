# Palindrome Text Analyzer

Palindrome Text Analyzer is a small Python application that scans long texts and detects palindrome words and phrases.

The application can analyze books, articles or any plain text document and generate simple statistics.

## Features

- Text parsing
- Palindrome detection
- Frequency analysis
- Longest palindrome search
- Console report

## Example

```
Loaded text

demo_book.txt

Words scanned

3842

Palindromes found

level
madam
refer
rotator
racecar

Longest

rotavator

Total unique

17
```

## Modules

- Repository
- Parser
- Detector
- Statistics
- Report
